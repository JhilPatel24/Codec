# Chatbot for Customer Service

## Project Overview
This project is a **Chatbot for Customer Service** developed using Python and basic Natural Language Processing (NLP) techniques. The chatbot is designed to answer common customer queries such as order status, delivery information, return and refund policies, payment methods, and customer support details.

The goal of this project is to understand how rule-based chatbots work and how simple NLP techniques can be applied to create an interactive customer support system.

## Project Objectives
- Automate responses to frequently asked customer questions  
- Understand text preprocessing techniques  
- Learn keyword-based intent detection  
- Design a simple conversational chatbot flow  
- Gain hands-on experience with Python and NLP  

## Features
- Responds to greetings and exit messages  
- Handles order tracking with order ID validation  
- Provides delivery time information  
- Answers return and refund related queries  
- Displays supported payment methods  
- Shares customer support contact details  
- Maintains basic conversation context  
- Fallback response for unknown queries  

## Technologies Used
- **Programming Language:** Python  
- **Libraries:** Regular Expressions (`re`)  
- **Tools:** VS Code / Jupyter Notebook
