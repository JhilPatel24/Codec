# Speech-to-Text Transcription
- Create a simple tool that converts audio recordings into text.
- Use basic libraries for speech recognition (e.g., SpeechRecognition in Python).

## Overview
This project converts spoken audio into text using Python. 
It uses the SpeechRecognition library with Google Speech API to transcribe speech from an audio file.

## Features
- Converts audio (.wav) to text  
- Simple and beginner-friendly  
- Works in Google Colab  

## Technologies Used
- Python  
- SpeechRecognition  

## How to Run
1. Install dependencies  
2. Upload an audio file  
3. Run the transcription script
